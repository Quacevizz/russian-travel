# Проект: Путешествие по России
--------------------------------
## Обзор
Проект является третьей работой курса Веб-разработчик от Яндекс.Практикум  
На сайте использовались следующие технологии:
1. Резиновая и адаптивная вёрстки
2. Grid и flexbox
3. Функция repeat
4. Grid области
